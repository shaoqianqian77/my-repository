# 奖励的不可预期（随机）让行为最大化 - 斯金纳箱


当B.F.斯金纳用他的盒子做进一步的实验时，他发现**如果老鼠每次都得到同样的奖励，那么它只有在饥饿时才会拉动杠杆。** 

使老鼠拉动杠杆的次数最大化的方法是改变它得到的奖励。

**如果它不知道当它拉动杠杆时，它是会得到一个颗粒，还是一个也没有，还是几个颗粒，那么它就会一次又一次地拉动杠杆。** 它就会在心理上着迷。

这就是著名的随机奖励原则——奖励的不可预期让行为最大化。
