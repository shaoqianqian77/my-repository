3. 设定目标但没完成 - 5 MP
4. HP = 耐力值
	1. 低于 50% 获得 debuff = “摆烂 n 分钟”；玩家行动效率减半。
	2. 恢复途径：恰饭、咖啡、锻炼、Yoga nidra。
