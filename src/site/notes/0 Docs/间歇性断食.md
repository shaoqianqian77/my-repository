# 间歇性断食

间歇性断食（intermittent fasting）让人更健康、长寿、思维敏捷。

- 只在 8 h 时间窗口规律饮食，16 h 只喝白开水
- 在饮食窗口内，吃的精、吃的好、吃的均衡
